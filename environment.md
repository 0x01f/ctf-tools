## Python

* [Python](https://www.python.org/)

* pip 豆瓣源设置

  在 `~/.pip/` 目录下新建 `pip.ini`（Windows）或 `pip.conf`（Linux）文件，内容如下：

  ```
  [global]
  index-url = http://pypi.douban.com/simple
  trusted-host = pypi.douban.com
  [list]
  format=columns
  ```
